## 🛠️ How to Use Rufus

1. Plug your USB flash drive into your computer.

2. Double-click the Rufus `.exe` file to open the program. You do not need to install it.

3. In Rufus, under **Device**, select your USB drive.

4. Under **Boot selection**, click **SELECT** and find your Windows ISO file on your computer.

5. The tool will automatically choose the right settings for Partition scheme and Target system. Usually, you do not have to change this.

6. You can leave the Volume Label as is or set a name for your USB drive.

7. File System should be set to **NTFS** by default.

8. Click **START** to begin creating your bootable USB.

9. Rufus will warn you that the USB will be erased. Confirm to continue.

10. Wait for the process to complete. This may take several minutes.

11. When done, safely eject your USB drive.

You now have a bootable USB that can install Windows 10 or 11.

## 🔧 Useful Tips

- Make sure you back up any important data from your USB drive before you start.
- If Rufus suggests downloading extra files, such as Windows installation files, allow it to do so.
- If your PC uses UEFI, use the GPT partition scheme; for older BIOS systems, use MBR.
- Check if Secure Boot is disabled in your PC’s firmware settings if the USB does not boot.
- Running Rufus as administrator can prevent permission problems.

---

## ❓ Troubleshooting

- If Rufus cannot detect your USB drive, try reconnecting it to another USB port.
- If the process stops with an error, try formatting your USB drive with Windows before using Rufus.
- If your PC does not boot from the USB, check your BIOS or UEFI settings to make sure USB booting is enabled.
- Make sure the ISO file is not corrupted. Try downloading it again if you face problems.
